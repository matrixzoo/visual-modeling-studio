# Visual Modeling Studio (可视化建模工场)
正如其名称，是一个基于主流大数据计算平台构建的可视化建模平台，通过简单的拖拽和配置的形式，让建模人员能够快速地在大数据平台上实现数据处理、模型构建、任务调度。
